# PROJECT SYNAPSE — Primera versión

Prototipo estático del perfil profesional y académico del Mtro. José Orozco Núñez.

## Qué incluye

- Pantalla de arranque tipo terminal.
- Estética anime y ciencia ficción sin utilizar personajes protegidos.
- Hero profesional con espacio reservado para una ilustración personalizada.
- Historia dividida en episodios.
- Proyectos, investigación, docencia y tecnologías.
- Fondo animado de estrellas.
- Animaciones al hacer scroll.
- Panel conceptual del asistente SYN.
- Menú adaptable a dispositivos móviles.
- Frases rotativas.
- Easter egg con el código Konami.

## Personalización rápida

Los colores principales están al inicio de `styles.css`, dentro de `:root`.
El contenido se encuentra en `index.html`.
Las animaciones e interacciones están en `script.js`.
